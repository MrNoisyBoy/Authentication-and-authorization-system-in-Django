# Authentication-and-authorization-system-in-Django

## Описание
Собственная система аутентификации и авторизации с использованием JWT и ролевой моделью доступа.

## Архитектура
- **Аутентификация**: JWT токены
- **База данных**: SQLite
- **Авторизация**: Ролевая модель с гибкими правами доступа

2. Установите зависимости:
